Тестовое задние от Valverde.

Требуется реализовать слайдер.

Техническое задание:
- Из макета требуется сверстать только слайдер
- Слайдер листается только стрелками 
- Бесконечная прокрутка
- Крайние слайды выходят за границы экрана
- Одинаковый вид на десктопе и мобильных устройствах
